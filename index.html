Bahamas Election Calendar

Instructions

  1. Select General or Bye-Election.
  2. Click a field, then click a date in the calendar. Use Show ranges to visualize statutory windows. Overlaps render as a diagonal split.
  3. Business days exclude Sundays and public holidays. If a holiday falls on a weekend, its observed weekday is treated as the holiday for counting, and the weekend date counts only if it is the actual holiday (Saturday) rather than the observed weekday.
  4. Anchor→Latest Return limits are calendar days (Constitution).
  5. Reverse logic: if Polling Day is known, count back 26–30/31 business days to estimate Issue of Writ.
  6. Printing: Print Calendar makes one landscape page per month (colors preserved). Print Key Dates is portrait with the key dates panel only.

ProvisionCalculation Rule
